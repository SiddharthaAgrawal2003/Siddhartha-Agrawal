##  Problem Descriptions

### Problem-1: Calculator Using Class
This program demonstrates object-oriented programming by implementing a calculator using a class.

**Features:**
- Uses encapsulation and class-based design
- Supports the following operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Accepts two double values and an operation type as input
- Handles invalid operations and division by zero gracefully

**Concepts Used:**
- Classes and objects
- Conditional statements
- Exception handling
- Scanner-based input
